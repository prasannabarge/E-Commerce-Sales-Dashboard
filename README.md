# E-Commerce Sales Dashboard

An interactive Power BI dashboard for analyzing e-commerce sales performance with Python-based data processing and SQL analytics.

## About

This project provides comprehensive sales analytics for e-commerce businesses. It combines data cleaning, SQL-based analysis, and interactive Power BI visualizations to deliver actionable insights on revenue, customer behavior, and product performance.

## Tech Stack

**Python:** Pandas, NumPy, Matplotlib  
**SQL:** Joins, Aggregations, Window Functions  
**Power BI:** KPI cards, Slicers, Drill-down insights  
**Tools:** Jupyter Notebook, GitHub

## Project Structure

```
E-Commerce-Sales-Dashboard/
│
├── data/                   # Sales and customer datasets
├── scripts/                # Python scripts for data processing
├── sql/                    # SQL queries for analysis
└── README.md
```

## Features

- Interactive Power BI dashboard with real-time metrics
- Sales performance tracking by product, category, and region
- Customer segmentation and behavior analysis
- Revenue trends and forecasting
- SQL-based data transformation and aggregation
- Python scripts for data cleaning and preprocessing

## Getting Started

### Prerequisites
- Python 3.8+
- SQL database
- Power BI Desktop

### Installation

```bash
git clone https://github.com/prasannabarge/E-Commerce-Sales-Dashboard.git
cd E-Commerce-Sales-Dashboard
pip install pandas numpy matplotlib seaborn
```

## Usage

1. **Data Preparation**
   - Place your sales data in the `data/` folder
   - Run Python scripts in `scripts/` for data cleaning

2. **SQL Analysis**
   - Execute SQL queries from `sql/` folder
   - Generate aggregated metrics and KPIs

3. **Dashboard**
   - Open Power BI file for interactive visualizations
   - Explore sales trends, top products, and customer insights

## Dashboard Highlights

- **Sales Overview:** Total revenue, orders, and profit margins
- **Product Analysis:** Best-selling products and category performance
- **Customer Insights:** Customer demographics and purchase patterns
- **Time Trends:** Monthly/quarterly sales trends and seasonality
- **Geographic Analysis:** Sales distribution by region

## Key Insights

- Identify top-performing products and categories
- Track sales growth and seasonal patterns
- Analyze customer purchase frequency and lifetime value
- Monitor regional sales performance
- Optimize inventory based on demand trends
